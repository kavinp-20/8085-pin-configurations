📌 Overview

This repository contains detailed information about the 8085 Microprocessor Pin Configuration. The project was created for learning and educational purposes to understand the functions of various pins in the 8085 microprocessor.

I learned this topic through the Etalvis Learning Platform under the guidance of Daniel Raj Sir 👨‍🏫.

🧠 About 8085 Microprocessor

The Intel 8085 is an 8-bit microprocessor developed by Intel. It contains 40 pins used for communication with memory, input/output devices, interrupts, clock generation, and power supply.

✨ Features
8-bit microprocessor
16-bit address bus
8-bit data bus
64 KB memory addressing capability
Operates with +5V power supply
Supports interrupt handling and serial communication
